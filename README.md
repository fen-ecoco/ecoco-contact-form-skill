# ECOCO 智慧客服表單 (Contact Form)

這是一個為 ECOCO 打造的動態智能客服表單，專為提升用戶回報體驗與後端處理效率而設計。

## 功能特點 (Features)

* **品牌視覺一致性**：嚴格遵守 ECOCO 品牌規範 (藍色與橘色為主調)，提升品牌形象。
* **智慧表單邏輯**：根據不同「主要問題分類」動態顯示或隱藏對應的子表單，引導用戶填寫正確的資訊。
* **智能 FAQ 攔截配對**：自動分析用戶輸入內容，若屬於常見問題，會直接給予 ECOCO 官網的使用教學連結，以降低客服人力負擔。
* **帳號移轉自動化流程**：特別規劃「帳號登入/資料修改」流程，並加上明確的移轉聲明與驗證防呆，保護用戶點數權益。
* **本地化 API 串接**：
  * 第二層級動態選單，會先提供城市篩選 (解析自 API 下發地址)，再針對該城市進行站點的關鍵字檢索。
  * 支援直接讀取本地端的 `ecoco_stations.json` 站點清單。
* **嚴謹檔案上傳驗證**：支援本機圖片 (限 3 張) 及影片 (限 1 部)，同時嚴格限制每個檔案不得超過 5MB。
* **無縫後端整合 (Google Apps Script)**：內建 Webhook / Fetch API，能將用戶回報的問題、產生的工單號及上傳的 base64 檔案直接丟給 GAS (並可轉存 Google Drive)，供後續轉發 Gmail。

## 技術架構

* **語法**：純粹的 HTML5, CSS3, JavaScript (Vanilla JS)。
* **依賴性**：零第三方前端框架 (不需要 Vue, React 或 jQuery)，載入速度極快。
* **字體**：採用 Google Fonts - Noto Sans TC。

## 如何使用

由於本專案採行動態 `fetch` 本機端 `.json` 檔案來實作：

1. **若擁有伺服器環境或開發環境**：
   可透過如 VS Code 的 Live Server 開啟（產生虛擬 HTTP Server），即能正常讀取 `ecoco_stations.json` 站點資料而不受瀏覽器 CORS 網頁安全政策阻擋。

2. **部署建議**：
   請將 `ecoco_contact_form.html` 與相關資料檔案部署至企業指定的靜態網頁伺服器（S3, Render, Vercel 等皆可）。
